I've been struggling the last few weeks to get things moving.  It doesn't help that I'm staring down one of the ugliest storms I've ever seen a month after moving to a new state while trying to find out how much the movers broke.
When I started learning to code I was pretty sure I was going to finish Flatiron in about 4 and a half months.  I was doing well, I was on track, and I was moving along like a champ.  At some point I started sacrificing understanding for completion, and I turned into some kind of super powered robot that was churning out lessons in minutes.  *This is not the way things should be done!*  When I finally got to the end of the more advanced Ruby OOP concepts I was telling myself this is easy,  I'm good.  And then I tried to implement it.

Sometimes you need to slow down a bit and make sure you are really understanding the concepts you are learning.  Hopefully I can start to understand that better as I put it into practice.  I love to rip through a problem, but that's a good way to hit a wall.  At this point you can do one of two things, try to climb the wall, or go back and figure out how your path led you there in the first place.
